Workshop de HTML e CSS

Este workshop de HTML e CSS foi criado para ensinar os fundamentos da criação de páginas web. Durante o encontro, os participantes aprenderão como estruturar um site utilizando HTML e como melhorar sua aparência com CSS, aplicando cores, fontes e layouts. De forma prática e simples, os alunos irão desenvolver pequenas páginas e entender como funciona a base da construção de sites na internet.
